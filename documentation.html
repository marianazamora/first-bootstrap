Process for Skill List

I decided that the method I would use for adding and deleting skills would be through an array. The function works by parsing through the items in an array and creating div elements for each one of them. Underneath the list there is an input box where the user can add a new skill. If a new skill is added the script deletes the last item of the array and adds the new item to the beginning of the array. It then removes all the div elements (removechild()) and iterates the new array.

  1. First, I had to learn about arrays in javascript, including editing the array through pop and unshift.
  2. I learned how to obtain the value from the input box and add it to the array.
  3. Then realized the characters would stay in the same box and searched how to reset the value.
  4. Once I could edit the array and pull the information from the input box I had to learn how to show every item of the array in HTML. First I thought about using for i in array but after searching in stackoverflow I learned that forEach method was better.
  5. I followed this example about iterating with the forEach method
  6. If I tested the function in console.log it would show all the items in the array correctly in their own line, however, when I edited the inner html of a div element it would just show the last item from the array. I thought that my iteration had to also create new div elements for every item in the array. Therefore, I searched how to create new div elements from an array (second answer “updated answer using forEach”).
